# Resolve Color Grading Panel

**Resolve Color Grading Panel** is a professional color manipulation interface designed for DaVinci Resolve users. As a powerful **Resolve Color Correction Tool**, it provides precise controls for lift, gamma, gain, offset, saturation, and contrast adjustments. This **Resolve Color LUT Manager** lets you import, organize, preview, and apply custom 3D LUTs to clips or entire timelines. With **Resolve Color Scopes Suite**, you get real-time video scopes including waveform (RGB and luminance), vectorscope, histograms, false color, and zone display for broadcast-safe grading.

## Features

- **Primary Wheels**: The **Resolve Color Grading Panel** includes log and linear grading controls.
- **Shot Matching**: Use **Resolve Color Correction Tool** for automatic color balancing across multiple clips.
- **LUT Workflow**: As a **Resolve Color LUT Manager**, it supports 17-point and 33-point 3D LUTs.
- **Professional Scopes**: This **Resolve Color Scopes Suite** offers parade, histograms, and CIE charts.
